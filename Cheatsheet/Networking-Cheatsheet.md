# ⚡ Networking – Command Cheat Sheet  
This cheat sheet was created to capture useful commands for quick future reference

| Command  | Description |
| ------------- | ------------- |
| `nslookup <domain>` | Query DNS Servers to lookup information about domain names, IP addresses, and DNS records |
| `whois <domain>` | Retrieves registration details from public WHOIS database |
| `telnet /<filename.html> HTTP/1.1 Host : anything` | Access file from the web server |
| `telnet <hostname> <port>` | Connect to remote system over TCP/ip |
| `arp -a` | Viewing ARP Cache | 
| `tracert <domain/IP>` | Windows tool show the data packet path to target |
